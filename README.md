# CodeAlpha_BasicNetworkSniffer
Basic Network Sniffer using Python and Scapy
Basic Network Sniffer

Description

This project demonstrates a basic network sniffer using Python.

Features

- Displays Source IP Address
- Displays Destination IP Address
- Displays Protocol Information
- Simulates packet capture and analysis

Technologies Used

- Python
- Pydroid 3

Purpose

The objective of this project is to understand network traffic, packet structure, and protocol analysis concepts in cybersecurity.
